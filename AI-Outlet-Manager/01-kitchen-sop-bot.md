### 🥘 冬阴功汤（Tom Yum Goong）

**分类：** 汤类 | **辣度：** 可调 | **出餐时间：** 8-10 分钟

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 虾 | 5-6 只（中号） | 去壳留尾 |
| 香茅（Lemongrass） | 2 根 | 拍扁切段 |
| 南姜（Galangal） | 3-4 片 | 薄切 |
| 卡菲尔青柠叶（Kaffir Lime Leaves） | 3-4 片 | 撕碎 |
| 小辣椒（Bird's Eye Chili） | 2-4 只 | 根据辣度调整 |
| 蘑菇 | 5-6 朵 | 对半切 |
| 小番茄 | 3-4 粒 | 对半切 |
| 鱼露（Fish Sauce） | 1.5 汤匙 | |
| 青柠汁（Lime Juice） | 2 汤匙 | 最后加 |
| 辣椒酱（Nam Prik Pao） | 1 汤匙 | |
| 椰奶（可选） | 3 汤匙 | Cream 版本用 |
| 清水/高汤 | 400ml | |
| 香菜（Cilantro） | 少许 | 装饰用 |

#### 制作步骤

```
步骤 1：大火煮滚 400ml 高汤
步骤 2：加入香茅、南姜、青柠叶 → 中火煮 2 分钟（出香味）
步骤 3：加入蘑菇 → 煮 1 分钟
步骤 4：加入虾 → 煮到变色（约 2 分钟），不要过熟
步骤 5：加入辣椒酱（Nam Prik Pao）→ 搅拌均匀
步骤 6：加入鱼露 1.5 汤匙 → 调味
步骤 7：加入小番茄、小辣椒
步骤 8：关火 → 加入青柠汁 2 汤匙（关火后加，保酸味）
步骤 9：（Cream 版）加入椰奶 3 汤匙搅匀
步骤 10：装碗 → 撒香菜 → 出餐
```

#### Plating 标准

```
- 用深碗（标准汤碗）
- 虾摆在最上面，看得到
- 香菜放中间装饰
- 汤色：清版 = 红橙透明 / Cream 版 = 橙白色
- 配一碟白饭（另外上）
```

#### 常见错误 & 纠正

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 青柠汁太早加（酸味蒸发） | 关火后才加青柠汁 |
| 虾煮太久（变硬） | 虾变色就关火，最多 2 分钟 |
| 香料没拍扁（没香味） | 香茅一定要拍扁再切 |
| 汤太咸 | 鱼露分次加，边尝边调 |
| 不够辣 | 加小辣椒或辣椒酱，不要加辣椒粉 |

### 🍜 Pad Thai 泰式炒粿条

**分类：** 面类 | **辣度：** 不辣 | **出餐时间：** 6-8 分钟

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 河粉（Flat Rice Noodles） | 150g | 提前泡软 |
| 虾 | 4-5 只 | 或换鸡肉 |
| 鸡蛋 | 1 个 | |
| 豆芽 | 一把（50g） | |
| 韭菜（Garlic Chives） | 4-5 根 | 切段 |
| 豆腐干 | 30g | 小丁 |
| 花生碎 | 1 汤匙 | 装饰用 |
| Pad Thai 酱汁 | 2.5 汤匙 | 预调好的 |
| 油 | 1.5 汤匙 | |
| 青柠 | 1/4 个 | 配碟 |

#### Pad Thai 酱汁预调配方（大批量）

```
罗望子酱（Tamarind Paste）: 200g
鱼露: 100ml
椰糖 / 棕榈糖: 150g
水: 50ml
→ 小火煮到糖溶化，搅匀，放凉备用
```

#### 制作步骤

```
步骤 1：大火热锅 + 油
步骤 2：下虾 → 煎到两面变色 → 推到一边
步骤 3：打蛋入锅 → 快速炒散
步骤 4：加入豆腐干 → 炒 30 秒
步骤 5：加入河粉 → 翻炒均匀（不要压碎）
步骤 6：倒入 Pad Thai 酱汁 → 大火快炒，让河粉上色
步骤 7：加入豆芽、韭菜 → 翻炒 30 秒（保脆）
步骤 8：关火装盘
步骤 9：撒花生碎 → 配青柠角
步骤 10：出餐
```

#### Plating 标准

```
- 用平盘
- 河粉堆中间，虾摆上面
- 花生碎撒在顶部
- 青柠角放旁边
- 可配：小碟辣椒粉 + 鱼露 + 糖
```

### 🥩 泰式烤猪颈肉（Kor Moo Yang）

**分类：** 烧烤/小食 | **辣度：** 不辣（蘸酱辣） | **出餐时间：** 15-18 分钟

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 猪颈肉 | 200g | 切厚片（1cm） |
| 蒜 | 3 瓣 | 剁碎 |
| 香菜根（Cilantro Root） | 2 根 | 剁碎 |
| 蚝油 | 1 汤匙 | |
| 鱼露 | 1 汤匙 | |
| 糖 | 1 茶匙 | |
| 白胡椒粉 | 少许 | |
| 油 | 1 汤匙 | 腌制用 |

#### 蘸酱（Jaew 酱）

```
干辣椒粉: 1 汤匙
鱼露: 2 汤匙
青柠汁: 2 汤匙
糖: 1 茶匙
烤米粉（Khao Khua）: 1 汤匙
葱花: 1 汤匙
香菜碎: 少许
→ 全部拌匀即可
```

#### 制作步骤

```
步骤 1：猪颈肉 + 所有腌料 → 拌匀腌制（至少 30 分钟，最好隔夜）
步骤 2：烤炉预热（中大火）
步骤 3：猪颈肉上烤炉 → 每面烤 5-6 分钟
步骤 4：翻面再烤 5-6 分钟 → 表面微焦但里面多汁
步骤 5：取出静置 2 分钟
步骤 6：切片（斜切，约 1cm 厚）
步骤 7：装盘 + 蘸酱 + 生菜垫底
步骤 8：出餐
```

#### Plating 标准

```
- 用长方盘或木板
- 生菜/卷心菜丝垫底
- 猪颈肉切片排列整齐
- Jaew 酱用小碟另上
- 可配黄瓜片装饰
```

### 🌿 泰式罗勒炒肉饭（Pad Kra Pao / 打抛饭）

**分类：** 饭类 | **辣度：** 中辣（可调）| **出餐时间：** 6-8 分钟

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 猪肉碎 / 鸡肉碎 | 150g | 也可用牛肉 |
| 圣罗勒（Holy Basil） | 一大把（20g）| **必须用圣罗勒**，不能用甜罗勒 |
| 蒜 | 4 瓣 | 剁碎 |
| 小辣椒（Bird's Eye Chili） | 3-5 只 | 根据辣度 |
| 蚝油 | 1 汤匙 | |
| 生抽 | 1 汤匙 | |
| 老抽 | 1/2 茶匙 | 上色用 |
| 鱼露 | 1/2 汤匙 | |
| 糖 | 1 茶匙 | |
| 油 | 2 汤匙 | |
| 鸡蛋 | 1 个 | 煎荷包蛋配 |
| 白饭 | 1 碗 | 另外煎/蒸好 |

#### 制作步骤

```
步骤 1：先煎荷包蛋 → 边缘焦脆，蛋黄半熟 → 放旁边备用
步骤 2：大火热锅 + 油 2 汤匙
步骤 3：下蒜和辣椒碎 → 爆香 20 秒（别炒焦）
步骤 4：下肉碎 → 大火快炒散，炒到变色
步骤 5：加蚝油 + 生抽 + 老抽 + 鱼露 + 糖 → 炒匀
步骤 6：加 1-2 汤匙水 → 让酱汁裹住肉
步骤 7：关火 → 扔进圣罗勒，快速翻拌（利用余热）
步骤 8：白饭装盘 → 肉浇在饭上
步骤 9：荷包蛋放旁边
步骤 10：出餐（可配鱼露辣椒小碟）
```

#### Plating 标准

```
- 用大圆盘
- 白饭在一边（半个盘）
- 肉在另一边（可见罗勒叶）
- 荷包蛋压在肉饭上或放旁边
- 配小碟：鱼露 + 辣椒圈 + 蒜末
```

#### 常见错误 & 纠正

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 用甜罗勒代替圣罗勒（味道不对） | 必须用 Holy Basil（圣罗勒） |
| 罗勒炒太久变黑 | 关火后才加，利用余热 |
| 肉炒老了 | 大火快炒，不超过 3 分钟 |
| 没有荷包蛋 | 必配，边缘焦脆蛋黄半熟 |
| 酱汁太多太湿 | 酱汁要裹肉，不是汤 |

| 鸡肉煮老 | 切薄片，中火煮 5 分钟即可 |

### 🥗 青木瓜沙拉（Som Tam）

**分类：** 沙拉/开胃菜 | **辣度：** 中辣（可调）| **出餐时间：** 5-7 分钟

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 青木瓜丝 | 150g | 刨丝 |
| 小番茄 | 4 粒 | 对半切 |
| 长豆 | 2 根 | 切段 |
| 蒜 | 2 瓣 | |
| 小辣椒 | 2-4 只 | 根据辣度 |
| 花生碎 | 1 汤匙 | |
| 虾米（干虾）| 1 汤匙 | 泡软 |
| 鱼露 | 1.5 汤匙 | |
| 青柠汁 | 1.5 汤匙 | |
| 椰糖 | 1 汤匙 | |
| 罗望子汁 | 1 茶匙 | 可选 |

**工具：** 木臼 + 木杵（没有的话用大碗 + 擀面杖）

#### 制作步骤

```
步骤 1：木臼中放蒜 + 小辣椒 → 轻捣（不要打碎成泥）
步骤 2：加入长豆段 → 轻捣几下
步骤 3：加入虾米 + 花生碎 → 拌一下
步骤 4：加入椰糖 + 鱼露 + 青柠汁 + 罗望子汁 → 拌匀让糖溶化
步骤 5：加入小番茄 → 轻捣让番茄出汁
步骤 6：加入青木瓜丝 → 用木杵轻压 + 用勺子翻拌（让木瓜吸汁）
步骤 7：试味道 → 应该是酸甜辣咸平衡
步骤 8：装盘
步骤 9：顶部撒多一点花生碎
步骤 10：配生菜/卷心菜片 + 长豆 + 糯米饭
```

#### Plating 标准

```
- 用浅盘或小竹篮
- 木瓜丝堆中间
- 花生碎撒顶部
- 旁边配生菜叶、卷心菜、长豆生吃
- 可配一小碗糯米饭
```

#### 常见错误 & 纠正

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 木瓜丝切太粗/太细 | 标准 0.3cm 粗细 |
| 打得太碎（像泥）| "轻捣"让汁渗入，保留脆度 |
| 糖没溶化（吃到颗粒）| 先把糖和鱼露青柠拌匀 |
| 用熟木瓜（软）| 必须用青木瓜（硬脆） |
| 味道单一 | 酸甜辣咸要平衡，多试味道 |

### 🍛 玛莎曼咖喱（Massaman Curry）

**分类：** 咖喱类 | **辣度：** 不辣（温和）| **出餐时间：** 15-20 分钟（牛肉需预炖）

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 牛肉 / 鸡肉 | 150g | 牛肉需预炖到软 |
| 玛莎曼咖喱酱 | 2 汤匙 | |
| 椰奶 | 250ml | 分两次加 |
| 马铃薯 | 1 个（中）| 切块 |
| 洋葱 | 1/2 个 | 切块 |
| 腰果 / 花生 | 2 汤匙 | 炒香 |
| 肉桂棒 | 1 小段 | |
| 八角 | 1 个 | |
| 小豆蔻 | 2 粒 | 可选 |
| 鱼露 | 1 汤匙 | |
| 椰糖 | 1.5 汤匙 | |
| 罗望子汁 | 1 汤匙 | 酸味关键 |

#### 制作步骤

```
步骤 1：（牛肉版）牛肉先用水炖 1 小时到软 → 备用
步骤 2：中火热锅 → 倒入 100ml 椰奶（稠的部分）
步骤 3：加入玛莎曼咖喱酱 → 炒到出油出香（3 分钟）
步骤 4：加入肉桂、八角、小豆蔻 → 炒 30 秒
步骤 5：加入肉 → 翻炒均匀上色
步骤 6：倒入剩下椰奶 150ml + 一杯水
步骤 7：加入马铃薯 + 洋葱 → 煮 10-12 分钟到软
步骤 8：加鱼露 + 椰糖 + 罗望子汁 → 调味
步骤 9：加入腰果/花生 → 再煮 2 分钟
步骤 10：装碗 → 配白饭出餐
```

#### Plating 标准

```
- 用深碗
- 汤色：深橙红色（比绿咖喱深）
- 肉块、马铃薯、腰果可见
- 可撒一点炒香的腰果在顶部
- 配白饭另碟
```

#### 常见错误 & 纠正

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 牛肉硬 | 必须预炖 1 小时到软 |
| 马铃薯没熟 | 切小块 + 煮够 10 分钟 |
| 味道不够层次 | 酸（罗望子）甜（椰糖）咸（鱼露）必须都有 |
| 腰果不香 | 先单独炒香或烤过再加 |
| 用一般咖喱酱代替 | 必须用 Massaman 酱（味道完全不同）|

### 🥭 芒果糯米饭（Mango Sticky Rice）

**分类：** 甜品 | **辣度：** 不辣 | **出餐时间：** 5 分钟（糯米提前蒸好）

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 糯米（蒸熟）| 120g | 需提前蒸 |
| 熟芒果 | 1/2 个 | 选 Honey 或 Harumanis |
| 椰奶 | 80ml | 分"咸奶"和"甜奶" |
| 椰糖 / 白糖 | 2 汤匙 | |
| 盐 | 1/4 茶匙 | |
| 玉米淀粉 | 1/2 茶匙 | 勾芡椰奶顶部 |
| 烤白芝麻 / 绿豆 | 1 茶匙 | 装饰 |

#### 制作步骤

```
步骤 1：糯米提前浸泡 4 小时 → 蒸 30 分钟到软（量大时提前准备）
步骤 2：制作"甜椰奶汁"：50ml 椰奶 + 糖 2 汤匙 + 盐 1/4 茶匙 → 小火煮到糖溶化
步骤 3：把甜椰奶汁倒入蒸好的糯米 → 拌匀 → 焖 10 分钟入味
步骤 4：制作"咸顶椰奶"：30ml 椰奶 + 一撮盐 + 玉米淀粉水 → 小火搅到浓稠
步骤 5：芒果削皮 → 切片或切块
步骤 6：盘中堆一小堆糯米饭
步骤 7：芒果摆在糯米饭旁边
步骤 8：淋上咸顶椰奶（在糯米上）
步骤 9：撒烤芝麻或炸绿豆装饰
步骤 10：出餐（可配一小杯椰奶另上）
```

#### Plating 标准

```
- 用浅盘或木板
- 糯米堆一小堆（半圆形）
- 芒果切片摆扇形或小方块
- 咸椰奶淋在糯米上（不是芒果）
- 芝麻撒顶部
- 可配新鲜花装饰（食用花）
```

#### 常见错误 & 纠正

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 糯米不软 / 太硬 | 浸泡 4 小时 + 蒸 30 分钟 |
| 椰奶直接淋没调味 | 必须分"甜奶"(拌饭) + "咸奶"(淋顶) |
| 芒果不够熟 | 选熟透的芒果，不熟会酸 |
| 糯米太湿 | 椰奶汁要控量，焖10分钟让饭吸收 |
| 冷掉再出餐 | 糯米要温热上桌（太冷会硬）|
