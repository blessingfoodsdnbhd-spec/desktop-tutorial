### 🥘 冬阴功汤（Tom Yum Goong）

**分类：** 汤类 | **辣度：** 可调 | **出餐时间：** 8-10 分钟

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 虾 | 5-6 只（中号） | 去壳留尾 |
| 香茅（Lemongrass） | 2 根 | 拍扁切段 |
| 南姜（Galangal） | 3-4 片 | 薄切 |
| 卡菲尔青柠叶（Kaffir Lime Leaves） | 3-4 片 | 撕碎 |
| 小辣椒（Bird's Eye Chili） | 2-4 只 | 根据辣度调整 |
| 蘑菇 | 5-6 朵 | 对半切 |
| 小番茄 | 3-4 粒 | 对半切 |
| 鱼露（Fish Sauce） | 1.5 汤匙 | |
| 青柠汁（Lime Juice） | 2 汤匙 | 最后加 |
| 辣椒酱（Nam Prik Pao） | 1 汤匙 | |
| 椰奶（可选） | 3 汤匙 | Cream 版本用 |
| 清水/高汤 | 400ml | |
| 香菜（Cilantro） | 少许 | 装饰用 |

#### 制作步骤

```
步骤 1：大火煮滚 400ml 高汤
步骤 2：加入香茅、南姜、青柠叶 → 中火煮 2 分钟（出香味）
步骤 3：加入蘑菇 → 煮 1 分钟
步骤 4：加入虾 → 煮到变色（约 2 分钟），不要过熟
步骤 5：加入辣椒酱（Nam Prik Pao）→ 搅拌均匀
步骤 6：加入鱼露 1.5 汤匙 → 调味
步骤 7：加入小番茄、小辣椒
步骤 8：关火 → 加入青柠汁 2 汤匙（关火后加，保酸味）
步骤 9：（Cream 版）加入椰奶 3 汤匙搅匀
步骤 10：装碗 → 撒香菜 → 出餐
```

#### Plating 标准

```
- 用深碗（标准汤碗）
- 虾摆在最上面，看得到
- 香菜放中间装饰
- 汤色：清版 = 红橙透明 / Cream 版 = 橙白色
- 配一碟白饭（另外上）
```

#### 常见错误 & 纠正

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 青柠汁太早加（酸味蒸发） | 关火后才加青柠汁 |
| 虾煮太久（变硬） | 虾变色就关火，最多 2 分钟 |
| 香料没拍扁（没香味） | 香茅一定要拍扁再切 |
| 汤太咸 | 鱼露分次加，边尝边调 |
| 不够辣 | 加小辣椒或辣椒酱，不要加辣椒粉 |

### 🍜 Pad Thai 泰式炒粿条

**分类：** 面类 | **辣度：** 不辣 | **出餐时间：** 6-8 分钟

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 河粉（Flat Rice Noodles） | 150g | 提前泡软 |
| 虾 | 4-5 只 | 或换鸡肉 |
| 鸡蛋 | 1 个 | |
| 豆芽 | 一把（50g） | |
| 韭菜（Garlic Chives） | 4-5 根 | 切段 |
| 豆腐干 | 30g | 小丁 |
| 花生碎 | 1 汤匙 | 装饰用 |
| Pad Thai 酱汁 | 2.5 汤匙 | 预调好的 |
| 油 | 1.5 汤匙 | |
| 青柠 | 1/4 个 | 配碟 |

#### Pad Thai 酱汁预调配方（大批量）

```
罗望子酱（Tamarind Paste）: 200g
鱼露: 100ml
椰糖 / 棕榈糖: 150g
水: 50ml
→ 小火煮到糖溶化，搅匀，放凉备用
```

#### 制作步骤

```
步骤 1：大火热锅 + 油
步骤 2：下虾 → 煎到两面变色 → 推到一边
步骤 3：打蛋入锅 → 快速炒散
步骤 4：加入豆腐干 → 炒 30 秒
步骤 5：加入河粉 → 翻炒均匀（不要压碎）
步骤 6：倒入 Pad Thai 酱汁 → 大火快炒，让河粉上色
步骤 7：加入豆芽、韭菜 → 翻炒 30 秒（保脆）
步骤 8：关火装盘
步骤 9：撒花生碎 → 配青柠角
步骤 10：出餐
```

#### Plating 标准

```
- 用平盘
- 河粉堆中间，虾摆上面
- 花生碎撒在顶部
- 青柠角放旁边
- 可配：小碟辣椒粉 + 鱼露 + 糖
```

### 🥩 泰式烤猪颈肉（Kor Moo Yang）

**分类：** 烧烤/小食 | **辣度：** 不辣（蘸酱辣） | **出餐时间：** 15-18 分钟

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 猪颈肉 | 200g | 切厚片（1cm） |
| 蒜 | 3 瓣 | 剁碎 |
| 香菜根（Cilantro Root） | 2 根 | 剁碎 |
| 蚝油 | 1 汤匙 | |
| 鱼露 | 1 汤匙 | |
| 糖 | 1 茶匙 | |
| 白胡椒粉 | 少许 | |
| 油 | 1 汤匙 | 腌制用 |

#### 蘸酱（Jaew 酱）

```
干辣椒粉: 1 汤匙
鱼露: 2 汤匙
青柠汁: 2 汤匙
糖: 1 茶匙
烤米粉（Khao Khua）: 1 汤匙
葱花: 1 汤匙
香菜碎: 少许
→ 全部拌匀即可
```

#### 制作步骤

```
步骤 1：猪颈肉 + 所有腌料 → 拌匀腌制（至少 30 分钟，最好隔夜）
步骤 2：烤炉预热（中大火）
步骤 3：猪颈肉上烤炉 → 每面烤 5-6 分钟
步骤 4：翻面再烤 5-6 分钟 → 表面微焦但里面多汁
步骤 5：取出静置 2 分钟
步骤 6：切片（斜切，约 1cm 厚）
步骤 7：装盘 + 蘸酱 + 生菜垫底
步骤 8：出餐
```

#### Plating 标准

```
- 用长方盘或木板
- 生菜/卷心菜丝垫底
- 猪颈肉切片排列整齐
- Jaew 酱用小碟另上
- 可配黄瓜片装饰
```

### 🌿 泰式罗勒炒肉饭（Pad Kra Pao / 打抛饭）

**分类：** 饭类 | **辣度：** 中辣（可调）| **出餐时间：** 6-8 分钟

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 猪肉碎 / 鸡肉碎 | 150g | 也可用牛肉 |
| 圣罗勒（Holy Basil） | 一大把（20g）| **必须用圣罗勒**，不能用甜罗勒 |
| 蒜 | 4 瓣 | 剁碎 |
| 小辣椒（Bird's Eye Chili） | 3-5 只 | 根据辣度 |
| 蚝油 | 1 汤匙 | |
| 生抽 | 1 汤匙 | |
| 老抽 | 1/2 茶匙 | 上色用 |
| 鱼露 | 1/2 汤匙 | |
| 糖 | 1 茶匙 | |
| 油 | 2 汤匙 | |
| 鸡蛋 | 1 个 | 煎荷包蛋配 |
| 白饭 | 1 碗 | 另外煎/蒸好 |

#### 制作步骤

```
步骤 1：先煎荷包蛋 → 边缘焦脆，蛋黄半熟 → 放旁边备用
步骤 2：大火热锅 + 油 2 汤匙
步骤 3：下蒜和辣椒碎 → 爆香 20 秒（别炒焦）
步骤 4：下肉碎 → 大火快炒散，炒到变色
步骤 5：加蚝油 + 生抽 + 老抽 + 鱼露 + 糖 → 炒匀
步骤 6：加 1-2 汤匙水 → 让酱汁裹住肉
步骤 7：关火 → 扔进圣罗勒，快速翻拌（利用余热）
步骤 8：白饭装盘 → 肉浇在饭上
步骤 9：荷包蛋放旁边
步骤 10：出餐（可配鱼露辣椒小碟）
```

#### Plating 标准

```
- 用大圆盘
- 白饭在一边（半个盘）
- 肉在另一边（可见罗勒叶）
- 荷包蛋压在肉饭上或放旁边
- 配小碟：鱼露 + 辣椒圈 + 蒜末
```

#### 常见错误 & 纠正

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 用甜罗勒代替圣罗勒（味道不对） | 必须用 Holy Basil（圣罗勒） |
| 罗勒炒太久变黑 | 关火后才加，利用余热 |
| 肉炒老了 | 大火快炒，不超过 3 分钟 |
| 没有荷包蛋 | 必配，边缘焦脆蛋黄半熟 |
| 酱汁太多太湿 | 酱汁要裹肉，不是汤 |

### 🐟 清蒸酸辣柠檬鱼（Steamed Hot & Sour Lime Fish / Pla Neung Manao）

**分类：** 鱼类/蒸 | **辣度：** 中辣（可调）| **出餐时间：** 15-18 分钟

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 全鱼（金目鲈 / 石斑 / 罗非鱼） | 500-600g | 鲜活最佳，剖腹洗净 |
| 青柠汁 | 4 汤匙 | 必须现榨 |
| 鱼露 | 3 汤匙 | |
| 椰糖 / 白糖 | 1.5 汤匙 | |
| 蒜 | 6-8 瓣 | 剁碎 |
| 小辣椒（Bird's Eye Chili） | 5-8 只 | 剁碎（辣度调整） |
| 香菜根 | 2 根 | 剁碎（重要！去腥增香） |
| 香菜叶 | 一把 | 装饰用 |
| 鸡汤 / 热开水 | 100ml | 稀释酱汁 |
| 青柠片 | 3-4 片 | 垫鱼身下 |
| 柠檬草段（可选） | 1 根 | 垫盘 |

#### 制作步骤

```
步骤 1：鱼身两面切 2-3 刀（让味道进去）→ 洗净擦干
步骤 2：蒸盘铺柠檬草 + 青柠片 → 鱼放上面
步骤 3：大火水滚后，鱼入蒸笼 → 蒸 10-12 分钟（500g）
     （鱼每多100g 多蒸 2 分钟）
步骤 4：同时制作酱汁：剁碎蒜 + 小辣椒 + 香菜根 → 放碗里
步骤 5：加入鱼露 3 汤匙 + 椰糖 1.5 汤匙
步骤 6：倒入热鸡汤 100ml → 搅到糖溶化
步骤 7：最后加青柠汁 4 汤匙 → 拌匀（试味：酸辣咸平衡）
步骤 8：鱼蒸好后 → 小心移到上菜盘（可倒掉蒸出的水）
步骤 9：酱汁均匀淋在鱼身上
步骤 10：撒大量香菜 → 出餐（趁热上桌）
```

#### Plating 标准

```
- 用长椭圆鱼盘（Whole Fish Plate）
- 鱼头朝左，鱼身完整
- 柠檬草/青柠片垫底（可见）
- 酱汁淋满鱼身
- 香菜叶铺满顶部（绿色点缀）
- 可加几片青柠角装饰
- 趁热上桌（冷掉腥味出来）
```

#### 常见错误 & 纠正

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 鱼蒸过头（肉柴）| 500g鱼 = 10-12分钟，不要超 |
| 蒸太短（没熟）| 眼睛变白、鱼鳍一拔就脱落 = 熟 |
| 青柠汁加早（蒸汽让酸味挥发）| 酱汁要在蒸好后才淋 |
| 没用香菜根（不够香）| 香菜根是灵魂，不能省 |
| 蒜和辣椒打太碎成泥 | 剁碎就好，保留颗粒感 |
| 酱汁没糖（太酸太咸）| 糖是平衡关键，一定要加 |
| 鱼不新鲜 | 鱼眼清亮、鳃红色 = 新鲜 |

### 🥗 青木瓜沙拉（Som Tam）

**分类：** 沙拉/开胃菜 | **辣度：** 中辣（可调）| **出餐时间：** 5-7 分钟

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 青木瓜丝 | 150g | 刨丝 |
| 小番茄 | 4 粒 | 对半切 |
| 长豆 | 2 根 | 切段 |
| 蒜 | 2 瓣 | |
| 小辣椒 | 2-4 只 | 根据辣度 |
| 花生碎 | 1 汤匙 | |
| 虾米（干虾）| 1 汤匙 | 泡软 |
| 鱼露 | 1.5 汤匙 | |
| 青柠汁 | 1.5 汤匙 | |
| 椰糖 | 1 汤匙 | |
| 罗望子汁 | 1 茶匙 | 可选 |

**工具：** 木臼 + 木杵（没有的话用大碗 + 擀面杖）

#### 制作步骤

```
步骤 1：木臼中放蒜 + 小辣椒 → 轻捣（不要打碎成泥）
步骤 2：加入长豆段 → 轻捣几下
步骤 3：加入虾米 + 花生碎 → 拌一下
步骤 4：加入椰糖 + 鱼露 + 青柠汁 + 罗望子汁 → 拌匀让糖溶化
步骤 5：加入小番茄 → 轻捣让番茄出汁
步骤 6：加入青木瓜丝 → 用木杵轻压 + 用勺子翻拌（让木瓜吸汁）
步骤 7：试味道 → 应该是酸甜辣咸平衡
步骤 8：装盘
步骤 9：顶部撒多一点花生碎
步骤 10：配生菜/卷心菜片 + 长豆 + 糯米饭
```

#### Plating 标准

```
- 用浅盘或小竹篮
- 木瓜丝堆中间
- 花生碎撒顶部
- 旁边配生菜叶、卷心菜、长豆生吃
- 可配一小碗糯米饭
```

#### 常见错误 & 纠正

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 木瓜丝切太粗/太细 | 标准 0.3cm 粗细 |
| 打得太碎（像泥）| "轻捣"让汁渗入，保留脆度 |
| 糖没溶化（吃到颗粒）| 先把糖和鱼露青柠拌匀 |
| 用熟木瓜（软）| 必须用青木瓜（硬脆） |
| 味道单一 | 酸甜辣咸要平衡，多试味道 |

### 🦐 三鲜青柠蒸海鲜饭（3 Seafood Lime Steamed with Rice）

**分类：** 饭类/蒸海鲜 | **辣度：** 中辣（可调）| **出餐时间：** 15-18 分钟

> 泰式青柠酱汁蒸三鲜（虾 + 鱿鱼 + 鱼肉），配白饭一起上。

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 大虾 | 4-5 只（去壳留尾）| |
| 鱿鱼 | 80g | 切圈或花刀 |
| 鱼肉（石斑/鲈鱼片） | 100g | 切厚片 |
| 青柠汁 | 3 汤匙 | 现榨 |
| 鱼露 | 2 汤匙 | |
| 椰糖 / 白糖 | 1 汤匙 | |
| 蒜 | 6 瓣 | 剁碎 |
| 小辣椒 | 4-6 只 | 剁碎 |
| 香菜根 | 2 根 | 剁碎 |
| 香菜叶 | 一把 | 装饰 |
| 热鸡汤 / 开水 | 80ml | |
| 柠檬草 | 1 根 | 切段垫盘 |
| 青柠片 | 3 片 | 装饰 |
| 白饭（蒸好的） | 1 碗（150g）| 另碟上 |

#### 制作步骤

```
步骤 1：海鲜洗净 → 虾开背去肠、鱿鱼切花刀、鱼切厚片
步骤 2：蒸盘铺柠檬草段 + 青柠片
步骤 3：海鲜排在盘上（鱼在下层，虾鱿鱼在上）
步骤 4：大火水滚后蒸 6-8 分钟（虾变红、鱿鱼卷起 = 熟）
     * 千万不能蒸过头，海鲜会老
步骤 5：同时制酱汁：剁碎蒜 + 辣椒 + 香菜根
步骤 6：加鱼露 2 汤匙 + 椰糖 1 汤匙 + 热鸡汤 80ml → 搅到糖溶
步骤 7：最后加青柠汁 3 汤匙 → 调酸辣咸平衡
步骤 8：海鲜蒸好 → 倒掉多余蒸汁（避免稀释酱汁）
步骤 9：酱汁均匀淋在海鲜上 → 撒大量香菜
步骤 10：出餐（白饭另碟，让客人自己配）
```

#### Plating 标准

```
- 用椭圆大盘（海鲜盘）
- 海鲜整齐排列：鱼铺底，鱿鱼圈中层，虾摆顶（红色点缀）
- 柠檬草、青柠片可见在底部
- 酱汁淋满
- 香菜叶铺在顶部（大量绿色）
- 白饭用小碗盛，另上
- 配小碟青柠角
```

#### 常见错误 & 纠正

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 三种海鲜一起蒸过久（虾鱿鱼老）| 大火 6-8 分钟即可，宁愿欠一点 |
| 海鲜不新鲜（有腥味） | 必须用当天新鲜海鲜，虾要活的 |
| 鱼肉切太薄（一蒸就散）| 切厚片（1.5cm），完整好看 |
| 鱿鱼没打花刀（卷不起来）| 内侧打十字花刀，熟了卷成花 |
| 酱汁太酸或太咸 | 先加糖溶化，再调青柠和鱼露 |
| 饭和海鲜一起装（饭吸汁变糊）| 白饭必须另碟上 |
| 没有香菜根 | 香菜根是泰式灵魂，一定要有 |

### 🥭 芒果糯米饭（Mango Sticky Rice）

**分类：** 甜品 | **辣度：** 不辣 | **出餐时间：** 5 分钟（糯米提前蒸好）

#### 食材清单（1份标准）

| 食材 | 分量 | 备注 |
|------|------|------|
| 糯米（蒸熟）| 120g | 需提前蒸 |
| 熟芒果 | 1/2 个 | 选 Honey 或 Harumanis |
| 椰奶 | 80ml | 分"咸奶"和"甜奶" |
| 椰糖 / 白糖 | 2 汤匙 | |
| 盐 | 1/4 茶匙 | |
| 玉米淀粉 | 1/2 茶匙 | 勾芡椰奶顶部 |
| 烤白芝麻 / 绿豆 | 1 茶匙 | 装饰 |

#### 制作步骤

```
步骤 1：糯米提前浸泡 4 小时 → 蒸 30 分钟到软（量大时提前准备）
步骤 2：制作"甜椰奶汁"：50ml 椰奶 + 糖 2 汤匙 + 盐 1/4 茶匙 → 小火煮到糖溶化
步骤 3：把甜椰奶汁倒入蒸好的糯米 → 拌匀 → 焖 10 分钟入味
步骤 4：制作"咸顶椰奶"：30ml 椰奶 + 一撮盐 + 玉米淀粉水 → 小火搅到浓稠
步骤 5：芒果削皮 → 切片或切块
步骤 6：盘中堆一小堆糯米饭
步骤 7：芒果摆在糯米饭旁边
步骤 8：淋上咸顶椰奶（在糯米上）
步骤 9：撒烤芝麻或炸绿豆装饰
步骤 10：出餐（可配一小杯椰奶另上）
```

#### Plating 标准

```
- 用浅盘或木板
- 糯米堆一小堆（半圆形）
- 芒果切片摆扇形或小方块
- 咸椰奶淋在糯米上（不是芒果）
- 芝麻撒顶部
- 可配新鲜花装饰（食用花）
```

#### 常见错误 & 纠正

| ❌ 常见错误 | ✅ 正确做法 |
|------------|------------|
| 糯米不软 / 太硬 | 浸泡 4 小时 + 蒸 30 分钟 |
| 椰奶直接淋没调味 | 必须分"甜奶"(拌饭) + "咸奶"(淋顶) |
| 芒果不够熟 | 选熟透的芒果，不熟会酸 |
| 糯米太湿 | 椰奶汁要控量，焖10分钟让饭吸收 |
| 冷掉再出餐 | 糯米要温热上桌（太冷会硬）|
