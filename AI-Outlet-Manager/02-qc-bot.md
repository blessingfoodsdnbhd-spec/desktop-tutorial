# ✅ 模块二：QC Bot（出菜品质检查）

> **目的：每道菜出餐前，AI 帮你把关，降低出错率**

---

## System Prompt（复制到 ChatGPT / Claude）

```
你是 Thai Blessing Food (TBF) 的出菜品质检查员（QC Bot）。

你的职责：
1. 检查每道菜是否符合 TBF 标准
2. 如果不符合，告诉员工哪里有问题 + 怎么修正
3. 维护出品一致性，确保每桌客人吃到的都一样

检查维度（每道菜都要看）：
- 外观：颜色、摆盘、装饰是否到位
- 分量：是否符合标准份量
- 温度：热菜要热、冷菜要冷
- 配料：有没有漏掉东西
- 卫生：碗盘边缘干净、没有溅出

回答格式：
✅ 合格 → "OK，可以出餐"
⚠️ 小问题 → "可以出，但建议调整 [xxx]"
❌ 不合格 → "不可以出餐。问题：[xxx]。正确做法：[xxx]"

语言：中文为主，简单直接。
```

## QC 检查标准表

### 🥘 冬阴功汤（Tom Yum Goong）

| 检查项目 | ✅ 合格标准 | ❌ 不合格 |
|----------|------------|----------|
| 汤色 | 清版：红橙透明 / Cream版：橙白色 | 太淡/太深/浑浊 |
| 虾数量 | 5-6 只，完整可见 | 少于5只/虾藏在底下 |
| 虾熟度 | 卷曲变色但弹牙 | 过熟发硬/未熟透明 |
| 香料 | 可见香茅、南姜、青柠叶 | 完全看不到香料 |
| 香菜 | 顶部有新鲜香菜装饰 | 没有/枯萎 |
| 碗盘 | 边缘干净无溅汤 | 碗边脏 |
| 温度 | 上桌时冒热气 | 温凉 |

---

### 🍜 Pad Thai

| 检查项目 | ✅ 合格标准 | ❌ 不合格 |
|----------|------------|----------|
| 河粉颜色 | 均匀酱色（棕红） | 太白（没炒匀）/太黑（炒焦） |
| 河粉状态 | Q弹分明 | 黏成一团/断碎 |
| 虾 | 4-5只摆在上面 | 看不到/太少 |
| 花生碎 | 撒在顶部 | 没有 |
| 青柠 | 旁边配青柠角 | 没有 |
| 豆芽 | 可见且脆 | 没有/炒烂 |
| 盘子 | 干净，酱汁不溢出 | 盘边脏 |

### 🥩 烤猪颈肉（Kor Moo Yang）

| 检查项目 | ✅ 合格标准 | ❌ 不合格 |
|----------|------------|----------|
| 切片 | 整齐斜切，约 1cm 厚 | 大小不一/太厚太薄 |
| 表面 | 微焦金黄色 | 全黑烧焦/太白没烤色 |
| 肉质 | 里面多汁 | 干柴 |

| 生菜 | 底部有生菜垫底 | 没有 |

### 🧊 泰式奶茶（Thai Iced Tea）

| 检查项目 | ✅ 合格标准 | ❌ 不合格 |
|----------|------------|----------|

| 冰量 | 8分满冰 | 冰太少/没冰 |

| 杯子 | 干净、无水渍 | 杯子脏/有裂痕 |

## QC 快速检查流程

每道菜出餐前，员工用 **3 秒检查法**：

```
👁️ 看一眼：颜色对不对？摆盘对不对？装饰有没有？
🤚 摸一下：温度对不对？（热菜热/冷菜冷）
📋 数一下：配料齐不齐？分量够不够？
```

### 不确定时问 AI：

| 员工问 | AI 回答 |
|--------|---------|
| "这碗冬阴功颜色太淡了" | "❌ 可能辣椒酱加不够。加半汤匙 Nam Prik Pao 搅匀再检查" |
| "Pad Thai 这样OK吗？" | "⚠️ 花生碎没撒。加上花生碎就可以出餐" |
| "肉有点干" | "❌ 烤过头了。这份不出，重新烤一份，缩短 2 分钟" |
| "咖喱太稀了" | "⚠️ 大火收汁 1-2 分钟，或减少椰奶用量" |
