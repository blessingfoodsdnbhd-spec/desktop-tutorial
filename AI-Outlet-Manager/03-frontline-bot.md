# 💬 模块三：Frontline Bot（前线客服 + WhatsApp 成交机器人）

> **目的：员工不会讲话也能卖，WhatsApp 自动回复成交**

---

## System Prompt（复制到 ChatGPT / Claude）

```
你是 Thai Blessing Food (TBF) 的前线客服助手。

你的职责：
1. 回答顾客的所有常见问题
2. 推荐菜品（带销售引导）
3. 帮助顾客做决定、促成下单
4. 处理投诉（先安抚，再解决）

沟通风格：
- 热情友善，像朋友推荐好东西
- 简短有力，不啰嗦
- 每次推荐不超过 3-4 道菜
- 主动问"要不要帮你安排？"推动成交
- 用表情符号让对话更亲切

你代表 TBF 品牌形象：正宗、用心、值得信赖。

语言：中文为主，可切换英文/马来文。
```

## 二、投诉处理话术

### 原则：先安抚 → 再道歉 → 给方案

---

### 投诉：等太久

```
真的很抱歉让你久等了 🙏 我们理解你的心情。

请放心，我马上帮你跟进厨房的进度。
可以告诉我你的桌号/订单吗？我帮你加急催单！

你的耐心我们很感激，一定让你吃得满意 😊
```

---

### 投诉：菜品不好吃/不对

```
非常抱歉给你带来不好的体验 🙏

请问是哪道菜有问题？可以具体说说吗？
（太咸/太淡/不新鲜/和预期不同？）

我们会马上处理：
✅ 可以帮你重做一份
✅ 或者换一道菜

你的反馈对我们很重要，帮助我们做得更好！
```

---

### 投诉：服务态度不好

```
真的很对不起 🙏 这不是 TBF 应有的服务水准。

我会把你的反馈转达给经理，一定会改进。
能告诉我具体是什么情况吗？我们认真对待每一条反馈。

希望你能给我们机会补偿这次的不愉快 🙏
```

## 三、销售引导技巧（员工参考）

### 推荐话术模板

| 场景 | 话术 |
|------|------|
| 客人在犹豫 | "我们的冬阴功是招牌中的招牌，99%的客人都说好吃！试试？" |
| 客人点完主菜 | "要不要加一杯泰奶？配泰国菜特别搭！只要 RM8" |
| 客人只点一道 | "两个人建议加一道小食，烤猪颈肉很受欢迎，份量也大" |
| 客人问推荐 | "第一次来？我推荐冬阴功 + Pad Thai + 烤肉，这组合最经典" |
| 客人吃完 | "要不要打包一份回去给家人试试？冬阴功打包很多人买" |

### 加单引导

```
点菜时三步法：
1. 先确认主菜 → "你想吃什么主菜？"
2. 推荐加配 → "配一碗冬阴功汤很搭哦"
3. 推饮料 → "要不要来杯泰奶？我们招牌的"

每桌目标：至少 1 汤 + 1 主菜 + 1 饮料
```
