## System Prompt（复制到 ChatGPT / Claude）

```
你是 Thai Blessing Food (TBF) 的员工培训导师（AI Trainer）。

你的职责：
1. 引导新员工完成入职培训流程
2. 教会员工基本操作和 SOP
3. 测试员工是否掌握标准
4. 耐心回答任何问题，不嫌烦

培训风格：
- 像一个友善的前辈在教后辈
- 一步一步来，不要一次讲太多
- 每教完一个部分，问"明白了吗？有没有问题？"
- 多用例子和比喻
- 做对了要鼓励，做错了要纠正但不骂人

语言：中文为主，简单易懂。
```

## 一、新员工培训流程（Day 1）

### 上午（3小时）：认识 TBF

```
⏰ 09:00 - 09:30 | 欢迎 & 介绍
→ TBF 品牌故事
→ 团队介绍
→ 餐厅参观（厨房、前线、仓库）

⏰ 09:30 - 10:00 | 基本规矩
→ 准时上班（迟到扣分）
→ 穿制服、戴帽子、洗手
→ 手机收好（工作时间不能玩）
→ 卫生标准（随时保持整洁）

⏰ 10:00 - 10:30 | 安全须知
→ 厨房安全（刀具、热油、火源）
→ 灭火器位置
→ 急救箱位置
→ 紧急出口

⏰ 10:30 - 12:00 | 岗位培训（分厨房/前线）
→ 详见下方各岗位培训内容
```

### 下午（3小时）：实操练习

```
⏰ 13:00 - 14:30 | 跟班操作
→ 跟着资深员工做，AI 辅助查询
→ 有问题随时问 AI

⏰ 14:30 - 15:30 | 独立操作
→ 在指导下独立完成 2-3 个任务
→ AI 检查并给反馈

⏰ 15:30 - 16:00 | Day 1 测验
→ AI 出 10 题测验
→ 通过 = 可以上岗
→ 不通过 = 第二天再练习相应部分
```

## 二、厨房岗位培训

### Level 1：基本操作（Day 1-2）

AI 教学内容：

```
📚 课程 1：卫生标准
- 洗手步骤（进厨房前、接触生肉后、上厕所后）
- 食材分类存放（生熟分开）
- 切菜板颜色区分（红=生肉、绿=蔬菜、白=熟食）
- 清洁流程（台面、地板、工具）

📚 课程 2：食材认识
- 认识泰国常用香料（香茅、南姜、青柠叶、Thai Basil...）
- 认识常用酱料（鱼露、蚝油、辣椒酱、咖喱酱...）
- 食材保存方法（冷藏/冷冻/常温）

📚 课程 3：基本刀工
- 切片、切块、切丝、剁碎
- 安全使用刀具
- 不同菜品需要的切法

📚 课程 4：设备使用
- 炉灶开关和火力调节
- 烤炉操作
- 冷藏/冷冻设备温度检查
```

### Level 2：菜品制作（Day 3-5）

```
📚 课程 5：学做第一道菜 — Pad Thai
（选 Pad Thai 因为步骤相对简单）
→ AI 一步一步教
→ 员工跟着做
→ 做完 AI 检查（用 QC 标准）

📚 课程 6：学做第二道菜 — 冬阴功
→ 同上流程

📚 课程 7：学做第三道菜 — 绿咖喱
→ 同上流程

📚 课程 8：学做小食 — 烤猪颈肉、炸鸡翅
→ 同上流程
```

### Level 3：独立出菜（Day 6-7）

```
📚 课程 9：独立制作 + QC 自检
→ 员工独立做菜
→ 自己用 QC 标准检查
→ 经理最终确认

📚 课程 10：处理突发情况
→ 菜太咸/太淡怎么补救
→ 食材不够怎么替代
→ 高峰期怎么提速
```

## 三、前线岗位培训

### Level 1：基本服务（Day 1-2）

```
📚 课程 1：仪容仪表
- 制服整洁
- 微笑面对客人
- 说话语气友善

📚 课程 2：服务流程
Step 1: 客人进门 → "欢迎光临！请问几位？"
Step 2: 带位入座 → 递菜单
Step 3: 给客人 2-3 分钟看菜单
Step 4: "请问可以点菜了吗？"
Step 5: 记录点单（复述确认）
Step 6: "好的！请稍等，马上为你准备"
Step 7: 上菜时 → "这是你的 [菜名]，请慢用"
Step 8: 中途关心 → "味道还可以吗？需要加什么吗？"
Step 9: 客人要结账 → 快速算单
Step 10: "谢谢光临！欢迎下次再来！"

📚 课程 3：菜单熟悉
- 每道菜的名字、口味、辣度
- 哪些是招牌菜
- 哪些适合不吃辣的客人
- 过敏原提醒（虾、花生、椰奶等）
```

### Level 2：推荐 & 销售（Day 3-4）

```
📚 课程 4：推荐技巧
- 客人不知道点什么 → 推荐 3 道招牌
- 客人只点一道 → 建议加汤或小食
- 客人吃完 → 推荐饮料或甜品
- 多人聚餐 → 推荐套餐组合

📚 课程 5：常见问题回答
→ 参考 TBF-FAQ.md 的 20 题话术
→ 员工要背熟 TOP 10 常见问题

📚 课程 6：处理投诉
→ 参考 03-frontline-bot.md 的投诉处理话术
→ 原则：先安抚 → 再道歉 → 给方案
→ 严重投诉立刻叫经理
```

### Level 3：独立服务（Day 5）

```
📚 课程 7：独立服务一桌客人（全流程）
→ 从带位到送客
→ 经理旁观评估

📚 课程 8：高峰期应对
→ 同时服务 3-4 桌
→ 优先级判断
→ 和厨房沟通技巧
```

## 四、AI 测验题库

> 员工完成培训后，AI 出题测验。答对 80% 以上才算通过。

### 厨房测验（示例）

```
Q1: 冬阴功汤的青柠汁什么时候加？
A: 关火后加（保留酸味）

Q2: Pad Thai 的河粉炒之前要做什么？
A: 提前泡软

Q3: 虾最多煮几分钟？
A: 2 分钟（变色就好）

Q4: 烤猪颈肉切多厚？
A: 约 1cm，斜切

Q5: 绿咖喱的椰奶要分几次加？
A: 2 次（先炒酱，后加汤）

Q6: 切菜板红色是切什么？
A: 生肉

Q7: 鱼露太咸了怎么办？
A: 分次加，边尝边调

Q8: 出菜前要检查哪 3 样？
A: 外观、温度、分量

Q9: Thai Basil 什么时候加？
A: 关火后加（利用余热）

Q10: 香茅为什么要拍扁？
A: 让香味释放出来
```

### 前线测验（示例）

```
Q1: 客人进门第一句说什么？
A: "欢迎光临！请问几位？"

Q2: 客人说"有什么推荐"，你推荐哪3道？
A: 冬阴功 + Pad Thai + 烤猪颈肉

Q3: 客人说不吃辣，你推荐什么？
A: Pad Thai / 泰式炸鸡 / Massaman Curry

Q4: 客人投诉等太久，第一步做什么？
A: 先安抚 + 道歉，再帮忙催单

Q5: 客人问"你们虾新鲜吗"，怎么回答？
A: "每天新鲜采购的，绝对新鲜放心！"

Q6: 点完主菜后可以推什么？
A: 推荐加汤或饮料（冬阴功/泰奶）

Q7: 客人吃完怎么说？
A: "谢谢光临！欢迎下次再来！"

Q8: 严重投诉怎么处理？
A: 立刻叫经理处理

Q9: 客人问有没有 WiFi，怎么回答？
A: 告诉密码 / "有的，密码在菜单上"

Q10: 多人聚餐（6人以上）建议什么？
A: 建议提前 booking + 推荐套餐组合
```

| 频率 | 内容 |
