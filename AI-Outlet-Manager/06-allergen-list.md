# ⚠️ TBF 过敏原清单（Allergen Reference）

> **用途：** 客人问"我对XX过敏，可以吃什么？"时，员工 3 秒内给出正确答案。
> **重要：** 过敏是**医疗问题**，答错可能出人命。员工必须熟记这份清单。

---

## 一、什么是过敏原？（给员工看的）

**过敏原 = 某些食材会让部分人产生严重过敏反应**

轻则：皮肤痒、起疹子、肚子痛
重则：呼吸困难、休克、**死亡**（anaphylaxis）

### 客人一般会问的过敏原（Top 10）

| 过敏原 | 中文 | 常见程度 | 严重程度 |
|--------|------|----------|----------|
| Shellfish | 海鲜 / 贝类（虾、蟹、贝）| 🔥🔥🔥 非常常见 | ⚠️⚠️⚠️ 可致命 |
| Fish | 鱼类 | 🔥🔥 常见 | ⚠️⚠️⚠️ 可致命 |
| Peanuts | 花生 | 🔥🔥🔥 非常常见 | ⚠️⚠️⚠️ 可致命 |
| Tree Nuts | 坚果（腰果、杏仁、核桃）| 🔥🔥 常见 | ⚠️⚠️⚠️ 可致命 |
| Eggs | 蛋 | 🔥🔥 常见 | ⚠️⚠️ 严重 |
| Dairy / Milk | 乳制品 / 牛奶 | 🔥🔥 常见 | ⚠️⚠️ 严重 |
| Gluten / Wheat | 麸质 / 小麦 | 🔥🔥 常见 | ⚠️ 中等 |

**注解：**

- `***` 青木瓜沙拉含虾米（Dried Shrimp），可做成不含版本
- `****` 青咖喱鸡饭可选加煎蛋（Plus RM3），不加则不含蛋

### ⚠️ 臭豆特别提醒（虾仁臭豆炒冬粉）

**臭豆（Petai / Sataw）不是过敏原，但有独特气味，部分人不接受：**

- 会造成尿液有味道（正常，12-24小时后消失）
- 糖尿病/肾病患者需咨询医生
- 孕妇慎食（传统观念认为过多上火）
- 员工必须在点餐时告知"这道菜有臭豆，气味比较特别"

## 三、员工标准回答话术（背熟！）

### 场景 1：客人问"我对虾/海鲜过敏，可以吃什么？"

```
✅ 可以推荐：
- 🥩 烤猪颈肉 Kor Moo Yang
- 🍛 绿咖喱鸡 Green Curry
- 🌿 打抛饭 Pad Kra Pao（选鸡肉/猪肉/牛肉版）
- 🥥 椰奶鸡汤 Tom Kha Gai
- 🐟 清蒸柠檬鱼（如果不对鱼过敏）
- 🥭 芒果糯米饭

❌ 不推荐（含虾/海鲜）：
- 冬阴功汤、Pad Thai（默认版本）
- 三鲜蒸海鲜饭
- 青木瓜沙拉（含虾米，可要求去除）

话术：
"您对海鲜过敏，我推荐鸡肉或猪肉版的菜品，
比如烤猪颈肉、绿咖喱鸡、打抛鸡饭都很受欢迎。
我会特别跟厨房说明，确保使用分开的锅具！"
```

---

### 场景 2：客人问"我对花生过敏"

```
✅ 可以吃：
- 大部分菜品都可以
- 冬阴功、绿咖喱、椰奶鸡汤、柠檬鱼、打抛饭、烤猪颈肉

❌ 要避免/要求去除：
- 🍜 Pad Thai（含花生碎）→ 可要求"no peanuts"
- 🥗 青木瓜沙拉（含花生碎）→ 可要求"no peanuts"

⚠️ 警告客人：
厨房可能有花生交叉污染，严重过敏者请谨慎

话术：
"您对花生过敏，Pad Thai 和青木瓜沙拉都有花生，
我们可以做不加花生的版本，但如果是严重过敏，
我们会特别严格处理，让厨房用专用工具。"
```

---

### 场景 3：客人说"我不吃鱼露"（穆斯林/素食/过敏）

```
⚠️ 泰国菜几乎所有菜都有鱼露（Fish Sauce）

✅ 可以做调整的菜：
- 🌿 打抛饭 → 可用生抽代替鱼露
- 🍜 Pad Thai → 可用素酱汁调整
- 🥩 烤猪颈肉 → 腌料可调整
- 🥭 芒果糯米饭 → 本来就不含鱼露

❌ 很难改（鱼露是灵魂）：
- 冬阴功、绿咖喱、椰奶鸡汤、青木瓜沙拉、清蒸鱼

话术：
"如果完全不能吃鱼露，我推荐打抛饭或 Pad Thai
可以用生抽替代。但像冬阴功、咖喱这些味道会
变很多。您可以接受改版吗？"
```

---

### 场景 4：客人问"我乳糖不耐 / 对椰奶过敏"

```
⚠️ 椰奶≠乳制品，但有些人对椰子过敏

❌ 含椰奶/椰子：
- 🍛 绿咖喱鸡
- 🥥 椰奶鸡汤（Tom Kha）
- 🥭 芒果糯米饭
- 冬阴功 Cream 版本

✅ 不含椰奶：
- 冬阴功清版、Pad Thai、烤猪颈肉
- 打抛饭、柠檬鱼、三鲜蒸海鲜、青木瓜沙拉

话术：
"椰奶版本我们有几道：绿咖喱、Tom Kha、
糯米饭。您可以选清版冬阴功、Pad Thai、
或打抛饭，这些都不含椰奶。"
```

---

### 场景 5：客人问"我是严重过敏（Severe Allergy）"

```
🚨 严重过敏 = 可能致命 = 必须非常谨慎

员工标准流程：
1. 先确认：是哪种过敏？多严重？
2. 告知经理 / 厨房长（必须！）
3. 明确告知客人：
   "我们厨房所有食材共用，可能有交叉污染。
    如果您的过敏是致命等级，我建议您考虑
    是否安全用餐。"
4. 如果客人坚持要点，厨房必须：
   - 用专用锅具和工具
   - 单独清洗台面
   - 主管亲自监督

话术：
"非常感谢您告诉我们。为了您的安全，我会立刻
通知经理和厨房主管。可以告诉我您的过敏程度
和紧急药物（如 EpiPen）位置吗？我们会特别
小心处理您的餐点。"
```

## 四、无法提供保证的情况（员工必讲）

**TBF 厨房属于共用厨房（Shared Kitchen）：**

```
必须告诉严重过敏客人：

"我们厨房有处理多种食材，虽然会尽量避免，
但无法 100% 保证完全没有交叉污染。
如果您的过敏是致命等级，请谨慎决定。"
```

这是**法律和道德**都要求的声明，保护客人也保护餐厅。

## 五、特殊饮食需求速查

| 饮食需求 | 可吃 | 不可吃 |
|----------|------|--------|
| **素食者（Vegetarian）** | 青木瓜沙拉（去虾米）、芒果糯米饭 | 大部分菜含肉/鱼露 |
| **纯素（Vegan）** | 仅芒果糯米饭（确认椰奶纯植物）| 几乎所有含鱼露的菜 |
| **穆斯林（Halal）** | 所有非猪肉菜 + 确认来源 | 烤猪颈肉、含米酒的菜 |
| **无麸质（Gluten-Free）** | 大部分（除 Pad Thai 酱汁 + 部分酱油）| 确认酱油类 |
| **孕妇** | 煮熟菜都可以 | 生鱼、半熟蛋、超辣 |
| **儿童** | 不辣菜：Pad Thai、椰奶鸡汤、糯米饭 | 小辣椒菜品 |

## 六、员工培训测验

> 经理每月抽查，员工必须 100% 答对。

1. 客人对虾过敏，能吃 Pad Thai 吗？
   **答：❌ 不能（Pad Thai 默认有虾 + 花生 + 蛋）**

2. 客人对花生过敏，可以点青木瓜沙拉吗？
   **答：✅ 可以，但要求"no peanuts"，厨房用干净工具**

3. 所有泰国菜都含什么常见过敏原？
   **答：鱼露（Fish Sauce）= 含鱼**

4. 客人说"severe shellfish allergy"，第一步做什么？
   **答：立刻通知经理/厨房主管，并告知客人交叉污染风险**

5. Tom Kha（椰奶鸡汤）含哪些过敏原？
   **答：椰奶 + 鱼露（微量鱼）**

6. 三道绝对不含海鲜的菜？
   **答：烤猪颈肉、绿咖喱鸡、打抛饭（选鸡/猪/牛）**

7. 客人说对椰子过敏，不能点哪 3 道？
   **答：绿咖喱、Tom Kha、芒果糯米饭**

8. 穆斯林客人能吃哪些 TBF 菜品？
   **答：除烤猪颈肉以外，所有菜都可以（但需确认食材来源 Halal）**

## 七、紧急情况处理

### 如果客人在店内出现过敏反应：

```
🚨 症状：皮肤红肿、呼吸困难、晕眩、呕吐

员工 5 步处理：

Step 1: 【立刻叫经理】+ 让客人停止进食
Step 2: 询问客人有没有紧急药（EpiPen / 抗过敏药）
Step 3: 如果有 → 协助使用 / 如果没有 → 立刻叫救护车
       🚑 马来西亚急救电话：999
Step 4: 让客人保持冷静 → 侧躺防止窒息
Step 5: 记录所有吃过的菜品 + 时间（给医生参考）

⚠️ 绝对不要：
- 给水喝（可能呛到）
- 让客人自己走（可能晕倒）
- 隐瞒或拖延通报
```

建议在菜单上标注图示：

```
🦐 = 含海鲜
🐟 = 含鱼/鱼露
🥜 = 含花生
🌰 = 含坚果
🥚 = 含蛋
🥛 = 含乳制品
🌾 = 含麸质
🫘 = 含大豆
🥥 = 含椰奶
🌶️ = 辣（1-3级）

例如：
🥘 冬阴功汤 🦐🐟🌶️🌶️🌶️ RM22
🍜 Pad Thai 🦐🐟🥜🥚🫘🌶️ RM18

```

让客人一眼看懂，减少员工解释时间 + 降低出错风险。
