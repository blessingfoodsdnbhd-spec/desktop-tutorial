## 一、成本概念快速扫盲

### 食材成本率（Food Cost %）怎么算？

```
食材成本率 = 食材成本 / 售价 × 100%

例：食材 RM11 ÷ 售价 RM20 = 55% ← 成本偏高
例：食材 RM7 ÷ 售价 RM20 = 35% ← 健康水平
```

### 55% 成本率的真实利润

```
售价 RM20
- 食材  RM11（55%）
- 人工  RM4   （20%）
- 租金  RM3   （15%）
- 水电等 RM1  （5%）
─────────────
= 净利 RM1   （5%）🚨

➡️ 卖 100 份才赚 RM100，不够给一个员工发一天工资。
```

### 35% 成本率的真实利润

```
售价 RM20
- 食材  RM7   （35%）
- 人工  RM4   （20%）
- 租金  RM3   （15%）
- 水电等 RM1  （5%）
─────────────
= 净利 RM5   （25%）✅

➡️ 卖 100 份赚 RM500，同样努力，多赚 5 倍。
```

## 六、成本控制行动方案

### 🟢 短期（1-2 周可做）

- [ ] 称一下每道菜实际用了多少食材（很多餐厅实际用量>SOP）
- [ ] 每道菜标准化分量，用量杯/电子秤
- [ ] 培训员工"不要多给"（一汤匙就是一汤匙）
- [ ] 每天盘点库存，找出损耗来源

### 🟡 中期（1-3 个月）

- [ ] 换供应商比价（至少询 3 家）
- [ ] 自制酱料（Pad Thai 酱、咖喱酱大批量做，省 20-30%）
- [ ] 菜单重整：下架成本过高、点单率低的菜
- [ ] 推广高利润菜（饮料、甜品、套餐）

### 🔴 长期（3-12 个月）

- [ ] 中央厨房统一备料（多店版）
- [ ] 食材联合采购（和其他餐厅合作议价）
- [ ] 季节性菜单（用便宜的时令食材做 special）
- [ ] POS 数据分析，淘汰点单率 < 5% 的菜品
