# 📸 TBF 菜品照片 Checklist

> **目的：** 每道菜都有一张标准照片，用于菜单、IG、FoodPanda、GrabFood、Google Business。
>
> **重要：** 好照片 = 多 30% 点击率 = 多 30% 订单。

---

## 一、拍照基础设置（通用）

### 📷 设备
- **手机：** iPhone 12 以上 或 Samsung S21 以上（够用）
- **专业：** Sony A7 / Canon 相机 + 50mm f/1.8 镜头（最佳）
- **三脚架：** 必备（防抖，多角度一致）

### 💡 灯光
```
✅ 最佳：自然光（窗边上午 9-11 点 / 下午 3-5 点）
✅ 备用：柔光灯 + 柔光伞（2 盏打 45 度角）
❌ 避免：直射阳光、暗黄室内灯、闪光灯
```

### 🎨 背景（统一风格最重要）
- **主色调：** 木色桌面 / 深色哑光桌面
- **配色：** 泰国风 — 藤编餐垫、香蕉叶、铜器、小竹篮
- **避免：** 杂乱桌面、品牌 logo 冲突物、手机/钥匙等

### 📐 拍摄角度（3 个标准角度）
1. **45 度角** — 最通用，菜单和 delivery 用
2. **俯拍（90 度）** — IG 网格照、平面设计用
3. **侧拍（0 度）** — 有层次的菜（汤、糯米饭）

### 1. 🥘 冬阴功汤（Tom Yum Goong）

```
✅ Hero 角度：45 度
✅ 碗：深红陶瓷汤碗（泰式）
✅ 重点突出：
   - 虾头朝上，虾身弯曲可见
   - 汤面蒸汽（刚上桌拍）
   - 香茅、青柠叶露出汤面
   - 红色辣椒油飘一点
   - 香菜绿色点缀
✅ 背景：木桌 + 香蕉叶垫
📸 Bonus：勺子舀起虾特写（Close-up）
```

### 2. 🍜 Pad Thai

```
✅ Hero 角度：45 度
✅ 盘：白色圆盘 或 藤编浅盘
✅ 重点突出：
   - 河粉颜色均匀（棕红）

✅ 背景：木桌 + 筷子（斜放）
```

## 四、Delivery 平台照片规范

### 🛵 FoodPanda / GrabFood 要求

| 规格 | 要求 |
|------|------|
| 尺寸 | 至少 1200 × 1200 px |
| 格式 | JPG / PNG |
| 背景 | 纯白 或 木色桌面 |
| 水印 | 不加（平台会标价）|
| 文件大小 | < 5MB |

### ❌ 平台常见拒图原因
- 照片模糊
- 背景太乱
- 有文字水印
- 光线太暗
- 菜品看不清（被其他物体遮挡）

## 六、建议一次性拍摄计划

### 🎬 一天拍完 15 道菜（高效方法）

```
准备阶段（提前一天）：
□ 联系摄影师 或 安排手机拍摄员
□ 准备拍摄道具（餐垫、筷子、小碟）
□ 确定拍摄场地（店内选有自然光的位置）
□ 打印拍摄清单给摄影师

拍摄日流程（上午 9am - 下午 4pm）：
09:00 - 09:30  设置灯光、背景、测试
09:30 - 12:00  拍引流款 6 道（菜量小先做）
   - Pad Thai, 打抛饭, 青木瓜沙拉
   - 青咖喱鸡饭, 冬阴功（小）, 椰奶鸡汤
12:00 - 13:00  午休（不休摄影师会累）
13:00 - 15:30  拍主菜 6 道
   - 烤猪颈肉, 绿咖喱鸡, 臭豆冬粉
   - 妈妈面, 清蒸鱼, 芒果糯米饭
15:30 - 16:30  拍 Premium 3 道（最后做 = 最新鲜）
   - 黄油河虾, 三鲜海鲜饭, 火山蟹肉饭
16:30 - 17:00  收尾、核对清单

预算参考：
- 专业摄影师（含修图）：RM 1,500 - 3,000 / 天
- 自己拍（手机）：RM 0（但质量差 30%）
- 建议：第一次请专业，之后自己补拍
```
