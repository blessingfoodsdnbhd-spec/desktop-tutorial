## 六、菜单格式版本建议

一份菜单做 **3 个版本**，不同用途：

### 📖 Version 1：印刷菜单（堂食用）

```
规格：A4 双面 × 2 张（共 4 页）
材质：铜版纸 250g + 覆膜防油
结构：
  P1 封面
  P2 开胃 + 汤品 + 咖喱
  P3 面类 + 饭类 + 海鲜
  P4 甜品 + 饮料 + 套餐 + 过敏原 + 联系方式

成本：RM 3-5 / 份（印 200 份）
```

### 📱 Version 2：电子菜单（WhatsApp / QR 扫码用）

```
规格：手机竖版 PDF 或图片
分辨率：1080 × 1920 px
结构：可多页滑动，每类一页
Bonus：菜品名可点击 → 跳转到 WhatsApp 下单
```

### 🛵 Version 3：外卖平台菜单（FoodPanda / GrabFood）

```
规格：每道菜一张 1200×1200 高清图
要求：
- 照片背景统一（白色或木色）
- 菜名简短（平台有字数限制）
- 描述 50 字以内
- 价格精准到 RM 0.01（如 RM 16.80 而非 RM 17）
```
